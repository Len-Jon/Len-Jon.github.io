南邮猫协小程序0成本解决方案

发表于 2023-09-10
原来从北大猫协小程序fork出来的南邮猫录小程序因成本原因被迫中止，因此，计划使用免费的Gitee Pages作为前端页面。

准备工作

环境清单

因为这是面向0基础的教程，若有一定的计算机基础，可对照此清单根据自己的情况跳过。实在不行，看左边侧边栏加我QQ，工作繁忙，有时间我会回的。

本文以市场占有率最高Windows系统为例，mac OS用户可以根据步骤寻找自己对应的教程。

至于Linux？你都用Linux了我想这些问题对你来说不是问题。

  • Git
  • 配置SSH密钥
  • 一个实名制Gitee账号（每个账号只能搭建一个站点，因此建议参考本解决方案的社团准备一个公共的Gitee账号，并且需要实名）或Github账号（好处是不用实名，坏处是国内可能会因为DNS污染无法访问）
  • Node.js与Hexo
  • Markdown编辑器

本文所提到的软件部分已上传到百度网盘。如果你只负责运维而不负责搭建，有一个取巧的办法，只需要下载一个Markdown编辑器，浏览本文Markdown编辑器相关部分，然后参考本文 搭建博客-我负责运维 的方案A。

Git

下载与安装

Git是一个版本管理工具，考虑到社团多人开发的需求，建议需要维护网站的都需要走一遍这个流程。首先先下载Git - Downloads (git-scm.com)，现在大家的电脑都是64位的吧？那么就选择这个傻瓜安装包，别选Windows Porable。

因为此链接为Github上的链接，所以也有可能下载失败，建议在我框起来的地方右键选择复制链接，然后用迅雷下载。还有最后一种方式：百度网盘（Git-2.42.0.2-64-bit.exe）

下载好了之后一路默认选项安装就可以了。安装完成后有个检查方式就是鼠标右键，会发现有个Open Git Bash Here。这个很重要，后面大多数操作都需要它完成。
